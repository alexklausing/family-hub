# Project Context: Family Hub

This project is a high-end family dashboard for an HP EliteDesk running Ubuntu 26.04 LTS (x86_64).

## Core Tech
- Laravel 11 (SQLite)
- Vue 3 (Composition API)
- shadcn-vue (Local components)
- Tailwind CSS v4

## Active Integrations
- Google Calendar
- Apple Calendar (CalDAV)
- Paprika Recipe Manager 3
- Amazing Marvin Tasks
- OpenWeatherMap API
